# llm_strongs_glosses

Per-language **Strong's-number → single-word gloss** tables, LLM-generated and
**anchored on the original-language lemma** (Hebrew/Greek), with the English gloss
used only as a hint. One file per language: `<lang>.tsv`.

## Why it's reusable
Strong's numbers are a language-independent key into the biblical lexicon. **Any
project that can get from a word to a Strong's number** (via an interlinear, an
aligned lexicon, or a tagged text) can borrow these glosses to render that concept
in 10+ languages — no per-language lexicon required.

## Format
TSV, columns:

| column | meaning |
|---|---|
| `strong` | Strong's number (e.g. `G4102`, `H2617`, extended like `G1161a`) |
| `lemma_ref` | original-language lemma reference |
| `en_ref` | English gloss (the hint used during generation) |
| `gloss` | the single most natural word in the target language |

Header line carries provenance: `# source=llm; anchored on the original lemma …`.

## Languages
`spa, rus, arb, hin, ben, asm, hau` — the gloss-thin languages. Codes are the
canonical **ISO 639-3 tag (BCP 47)**, one file `<code>.tsv`
(see [strongs/README.md](../strongs/README.md#language-codes)).
~19.5k codes per language (full Strong's-with-gloss universe); a tiny tail (~0.3%)
of codes is legitimately ungloss-able and omitted.
